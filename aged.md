# Agent Skills Directory

Based on the open agent skills ecosystem (skills.sh), here are the relevant skills curated for your project's Design, Frontend, Backend, and UI/UX needs:

## Design & UI/UX
- **`anthropics/skills/frontend-design`**: Core skill for creating beautiful, accessible, and responsive frontend designs. Focuses on modern UI/UX principles.
- **`design-system-architect`**: Helps establish and maintain a consistent design system, typography, and color palettes.
- **`tailwind-expert`**: For implementing modern, dynamic, and responsive UI layouts efficiently.

## Frontend
- **`vercel-labs/agent-skills/vercel-react-best-practices`**: Implements best practices for React, Next.js, and modern component-driven architectures.
- **`vercel-labs/agent-browser/agent-browser`**: Useful for automated end-to-end testing and verifying frontend UI renders correctly.
- **`frontend-performance-optimizer`**: Ensures the frontend is fast, accessible, and SEO-friendly.

## Backend
- **`api-design-standard`**: Best practices for structuring scalable RESTful and GraphQL APIs.
- **`database-architect`**: For designing efficient, scalable, and secure database schemas.
- **`backend-security-auditor`**: Scans and enforces security best practices for backend infrastructure and endpoints.

## General Workflow
- **`mattpocock/skills/grill-me`**: Excellent for aligning on plans through an interactive interview to resolve complex design and architecture decisions before coding.
- **`vercel-labs/skills/find-skills`**: To discover and install additional relevant skills as the project evolves.

---
*Note: To install official skills from the registry, you can use the command: `npx skills add <owner/repo>`*
