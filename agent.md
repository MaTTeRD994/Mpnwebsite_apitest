# Agent Instructions & Skills

This project is a **Notion-like personal workspace application** for notes, to-dos, and workflows. During development, the agent must strictly adhere to the following design, frontend, and UI/UX skills provided:

## 1. Design Taste Frontend (`leonxlnx/taste-skill/design-taste-frontend`)
- Emphasize high-quality, modern aesthetics that "wow" the user.
- Use harmonious color palettes (e.g., sleek dark modes, HSL tailored colors), proper modern typography (e.g., Inter, Roboto, Outfit), and smooth micro-animations.
- Create an interface that feels premium, state-of-the-art, and avoids feeling like a generic MVP.

## 2. UI/UX Pro Max (`nextlevelbuilder/ui-ux-pro-max-skill/ui-ux-pro-max`)
- Prioritize exceptional user experience: Intuitive navigation, clear visual hierarchy, and responsive layouts.
- Provide immediate visual feedback for user interactions (rich hover effects, active states, loading transitions).
- Ensure accessibility, smooth transitions, and logical flow between tasks (notes to workflows).

## 3. Web Design Guidelines (`vercel-labs/agent-skills/web-design-guidelines`)
- Maintain a minimalist, clean, and modern UI. Remove unnecessary visual clutter.
- Optimize for performance, fast load times, and semantic structure.
- Build reusable, focused components that align with a strict design system.

## Core Development Rules
- **Framework**: Use a robust framework like Next.js for the web app structure.
- **Styling**: Prioritize Vanilla CSS (for maximum flexibility and control) with rich aesthetics, glassmorphism, and dynamic animations. Avoid TailwindCSS unless explicitly confirmed.
- **No Placeholders**: Generate actual UI assets or use working demonstrations instead of empty placeholders.
